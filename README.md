# AI-agents-hackathon
A STTP hackathon on building AI Agents
# SmartHire AI

## Features

- Resume Screening
- Job Description Improvement
- Interview Question Generation
- Salary Benchmark Lookup
- Candidate Email Generation
- Human-in-the-Loop Confirmation

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- Pandas
- FAISS
- Sentence Transformers

## How to Run

pip install -r requirements.txt

streamlit run app.py
